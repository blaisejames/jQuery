Click

Click the button to binds an event handler to the "click" JavaScript event, or trigger that event on an element.

Hide

Click the button to hide the matched elements. In this case, the image to the right.

Show

Click the button to display the matched elements. In this case, an image on the right.

Toggle

Click the button twice: once to display and then to hide the matched elements. In this case, an image on the right.

Slide Down

Click the button to display the matched elements with a sliding motion. In this case, an image on the right.

Slide Up

Click the button to hide the matched elements with a sliding motion. In this case, the image on the right.

Slide Toggle

Click the button twice: once to hide and then to diplay the matched elements with a sliding motion. In this case, an image on the right.

Fade In

Click the button to display the matched elements by fading them to opaque. In this case, an image on the right.

Fade Out

Click the button to hide the matched elements by fading them to invisible. In this case, the image on the right.

Add Class

Click the button to add the specified class(es) to each element in the set of matched elements. In this case, a gray box around the image on the right.

Before

Click the button to insert a DOM element before each element in the set of matched elements. In this case, the word "roses" before the image of roses on the right.

After

Click the button to insert a DOM element after each element in the set of matched elements. In this case, a the word "roses" after the image on the right.

Append

Click the button to insert content, specified by the parameter, to the end of each element in the set of matched elements. In this case, the words at the end of the sentence on the right.

The lazy brown fox jumps over

Html

Click the button to get (or set) the HTML contents of the first element in the set of matched elements. In this case, see the html equivalent of the sentence on the right.

The lazy brown fox jumps over the lazy dog.

Attr

Click the button to get the value of an attribute. In this case, see the class name of the h5 attribute of the sentence on the right.

The lazy brown fox jumps over the lazy dog.

Val

Click the button to get the current value of the first element in the set of matched elements. In this case, the value inside the input box on the right.

Text

Click the button to get (or set) the combined text contents of each element in the set of matched elements. In this case, the sentence in the h6 tag on the right.

The five boxing wizards jump quickly.

Data

Click the button to store (or return) arbitrary data associated with the matched elements. In this case, clicking will store "pineapple" and "penguin" in the incomplete sentence on the right.
